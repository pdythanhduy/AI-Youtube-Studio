## Japanese Narration Naturalness Assessment

### Line-by-Line Review (problem lines only)

---

**Line (ACT I):**
```
以来、島全体が一企業によって管理される、前代未聞の「企業城下島」となりました。
```
**Issue [M2]:** 「企業城下島」は造語。「城下町」は馴染みのある語だが「城下島」は標準的な日本語用法ではない。ドキュメンタリーナレーションで造語が出ると視聴者の集中が途切れる。
**Fix:** 「以来、島全体がひとつの企業によって管理される、前代未聞の島となりました。」または「三菱が、島そのものを所有する体制となりました。」

---

**Line (ACT I):**
```
島民はここで生まれ、学び、働き、結婚し、子どもを産みました。端島は、彼らにとって、日本そのものでした。
```
**Assessment:** 自然で感情的に適切。「日本そのものでした」は少し大きいが、ドキュメンタリー的な締めとして機能する。問題なし。

---

**Line (ACT II):**
```
「軍艦島」という名前を、あなたはどこかで聞いたことがあるかもしれません。
```
**Assessment:** 自然。視聴者への直接的な問いかけとして適切。

---

**Line (ACT II):**
```
1939年から1945年にかけて、太平洋戦争の時代。朝鮮半島および中国大陸から連行された人々が、この炭坑の地下深くで働かされました。
```
**Assessment:** 「連行された」は事実として正確で、適切な語彙。「強制連行」は政治的に更に踏み込んだ語だが「連行された」は事実記述として使用可能。「働かされました」は受動態で事実を述べており適切。問題なし。

---

**Line (ACT II):**
```
端島が世界遺産に登録されたのは、明治の産業革命を象徴する遺産としてです。しかしその同じ場所で、同じ時代に、別の人々が異なる経験をしていた——そのことを、世界遺産という称号はどのように扱うべきでしょうか。
```
**Assessment:** 強力な問いかけ。「同じ時代に」は時代の重複を正確に示す（ただし技術的には世界遺産登録が評価するのは明治期産業化であり、強制労働は昭和期。時代がずれているが「同じ場所で」という表現で処理しているため許容範囲）。問題なし。

---

**Line (ACT III):**
```
家具は残されたまま。教科書は机の上に。子どもの玩具は、廊下の片隅に。
```
**Issue (minor):** 「玩具（おもちゃ）」は書き言葉として「玩具」と書いても問題ないが、ナレーションで読み上げる場合「おもちゃ」と読むべきか「がんぐ」と読むべきか、voice_script.txtに指示がない。
**Fix:** `voice_script.txt`の該当行にルビ注記を追加：「子どもの玩具（おもちゃ）は」

---

**Line (ACT III):**
```
コンクリートだけが残り、かつての生活の痕跡だけが——静かに、しかし確実に、失われていきました。
```
**Assessment:** 「静かに、しかし確実に」はやや文学的だが、dark_documentaryスタイルに合致。問題なし。

---

**Line (ACT IV):**
```
産業の誇りと、強制の記憶——どちらも、この島では本物の出来事でした。どちらかを選ぶ必要はないかもしれません。しかし、どちらも忘れることは——できないのかもしれません。
```
**Assessment:** ビデオ全体の核心文。バランスが取れている。どちらか一方を断罪せず、両側の歴史を認める構造は適切。「本物の出来事でした」は「real events」の直訳的表現だが文脈上機能する。問題なし。

---

**Line (OUTRO):**
```
この島が、あなたに問いかけているものは——何でしょうか。
```
**Assessment:** 適切な締め。japan_template要件（問いで終わる）を満たす。語順が少し倒置的だが文学的効果として有効。問題なし。
