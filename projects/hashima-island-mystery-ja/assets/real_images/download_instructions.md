## File Organization After Download

Place downloaded files in:
```
assets/
  real_images/
    downloaded/        ← verified licensed files go here
      S001/
      S007/
      S008/
      S020/
      S024/
    staging/           ← files pending legal review
    rejected/          ← files that failed review
```

For each downloaded file, create a companion `.txt` file:
```
filename: Battle-Ship_Island_Nagasaki_Japan.jpg.license.txt
content:
  Source: https://commons.wikimedia.org/wiki/File:Battle-Ship_Island_Nagasaki_Japan.jpg
  License: CC BY 2.0
  Attribution: kntrty / Wikimedia Commons / CC BY 2.0
  Downloaded: [date]
  Commercial use: YES
  SA required: NO
  Used in scenes: S007
```

This license audit trail is required for any future copyright dispute.
